# Rearranging-the-Eight-Figures
***
QT大法好，想用C++做图像用户界面的开发选它！(并支持多种跨平台语言开发)  

另：在QT里完成ui界面设计图之后项目对.ui程序是无法完成自动编译的，要在cmd里手动完成  
> 用的是uic.exe程序  
> 进入cmd后 cd 进入uic.exe所在目录  
> 执行命令（务必指明两个文件所在的目录）：  
> uic XXX.ui -o ui_XXX.h
